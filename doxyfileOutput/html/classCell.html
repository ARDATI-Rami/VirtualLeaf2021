◆ ConstructWalls()

void Cell::ConstructWalls ( void  )

Constructs Walls, but only one per cell boundary.

Standard method constructs a Wall for each cell wall element, making transport algorithms computationally more intensive than needed.

We can remove this? Well, let's leave it in the code in case we need it for something else. E.g. for importing leaf architectures in different formats than our own... :-)
